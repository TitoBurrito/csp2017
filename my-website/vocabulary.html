List of Vocabulary Words


  1. Sprites
    • A graphic that can be coded to do what the creator of a
      program wants and can be placed on top of a backdrop, or
      behind it.
  2. Scripts
    • A set of instructions or code that affects the sprites or
      the backdrops.
  3. Iteration
    • Repetition of a specific task multiple times to execute
      a goal.
  4. Integrated Development Environment (IDE)
    • A website or application that allows a creator to fully
      develop, run, and fix a program.
  5. Stage
    • The location where you place everything. Backdrops, sprites,
      costumes, sounds, and all code.
  6. Costumes
      7. Appearance of a sprite, or character.
  7. Sounds
    • Audio files that you can place whenever you want in a Scratch project.
  8. Pseudocode
    • A rotating programming language used in programming.
  9. Pair Program
    • A program made on one computer by a group or collaboration.
  10. See Inside
    • A button on any Scratch project that allows you to see all
      of the sprites, backdrops, and code.
  11. Event
    • The action that happens that then leads to another action
      or code to happen.
  12. Boolean Expression
    • Tells if a statement is true or false. This is mostly used
      in the IF structure.
  13. Execute
    • To run a program or instruction(s).
  14. Bug
    • a mistake in the programming that causes an error or break.
